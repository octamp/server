# Octamp Server

Octamp Server is scalable Websocket Server for PHP using Redis Pub/Sub

### Requirements

- PHP >= 8.1
- OpenSwoole
- Redis Server

### Example


Run the server
```shell
php example/server.php
```

Run the receiver client
```shell
php example/receiverClient.php
```

Run the sender Client
```shell
php example/senderClient.php
```
